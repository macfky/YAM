Si on arrive a défoncer les chauves
On pourrais s'en sortir

Défoncer les chauves

Ensemble, défoncons les chauves
Tant qu'a faire le toutr d'la ville

Oué, la sagesse la vermine
Leur mensonges qu'on exposent

Défoncer les chauves
